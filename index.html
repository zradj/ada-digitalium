The Team

Welcome to the website of “ADA Digitalium” — Team №28!

ADA Digitalium is a team of four ADA students (Zaur Rajabov, Said Jahangirov, Fikrat Rustamli, Samad Ganbarov) who cooperate to solve problems and accomplish tasks for the Principles of Information Systems course. So far, we have organized an Hour of Code event in one of Baku schools. Making this website is another task we were assigned to do. We hope you'll like it as much as we do!

The website uses various technologies, starting from basic HTML/CSS to more complex ones, such as JavaScript. We would like to thank the authors of Codecademy, which was very helpful in providing us with the knowledge on how to use these technologies.

Hour of Code

“Hour of Code” is an event which is organized thousands of times each year all around the world by people who aspire to spread the knowledge about Computer Science among children.

Organizers usually use free interactive courses on the Code.org website to teach kids. The topics covered usually include basic programming concepts, which are delivered through the use of “visual programming” in order to make it easier to understand these concepts for children.

“Hour of Code” is volunteer-based, meaning that anyone can organize this event. The venue can be a school, a university, another educational facility, or any other community.

“ADA Digitalium” held an Hour of Code event at the Baku Secondary School №25 with the 5th grade on November, 11th at 8 AM. There were roughly 30 students listening to us, and they were very participative in the event. All of them were very happy to see us.

We would like to thank the school's administration for giving us an opportunity to organize this event in their school, and we especially thank them for their cooperation and exceptional willingness to help us.

We strongly recommend others to organize a similar event to contribute to the society they are living in. We think children in these times should get more familiarized with programming and Information Technologies in general.

The Project

This website uses HTML5 (like almost any other website on the Web). Each section of the website corresponds to a separate HTML section. The website uses a wide variety of tags from basic <div> and <p> to semantic tags like <figure> and <section>. The whole content is placed in one file — namely, index.html — to prevent intra-website links. We believe this behavior is unnecessary for such a small website.

This website uses CSS3 to style the elements. Various technologies are used to deliver a more pleasant look to the viewer. For instance, this website uses Flexbox, which greatly simplifies positioning. CSS is written in the Atomic CSS style, i.e. frequently used styles are broken down into separate classes to ensure reusability and make the process of further modification of styles easier and less cumbersome.

Finally, this website utilizes JavaScript for animation and effects. For instance, JavaScript tracks scrollilng of the webpage and shrinks the navigation bar if the user scrolls below a particular mark. It also displays the “back to the top” button on the bottom-right in this scenario, which allows the user to quickly move to the beginning of the website.

The Team

Welcome to the website of “ADA Digitalium” — Team №28!

ADA Digitalium is a team of four ADA students (Zaur Rajabov, Said Jahangirov, Fikrat Rustamli, Samad Ganbarov) who cooperate to solve problems and accomplish tasks for the Principles of Information Systems course. So far, we have organized an Hour of Code event in one of Baku schools. Making this website is another task we were assigned to do. We hope you'll like it as much as we do!

The website uses various technologies, starting from basic HTML/CSS to more complex ones, such as JavaScript. We would like to thank the authors of Codecademy, which was very helpful in providing us with the knowledge on how to use these technologies.
